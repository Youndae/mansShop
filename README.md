# Man's Shop 개인 프로젝트

### 목적
* 시큐리티를 적용해 관리자와 사용자의 권한을 분리
* 결제 API 적용
#
### 사용기술
* Spring 5.0.7
* Maven
* MyBatis
* JSP
* Spring Security
* Lombok
* Oracle
* Tomcat 9
* CSS
* JQuery
* Ajax
* 사용 IDE  :  IntelliJ
